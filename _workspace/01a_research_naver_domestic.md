## 1. 네이버 스포츠 마이팀 서비스 현황

### 1.1 핵심 기능

네이버 스포츠 마이팀(MY팀)은 네이버앱 및 웹에서 사용자가 응원하는 스포츠 구단을 등록하고, 해당 팀의 경기 일정/결과/뉴스를 우선적으로 제공받는 개인화 기능이다.

#### 주요 기능 목록

| 기능 | 설명 |
|------|------|
| **MY팀 등록** | 종목별 응원 구단을 "+" 버튼으로 추가, 복수 팀 등록 가능 |
| **마이티켓** | 야구 팬 인증 기능. 발급 시 경기 페이지에 응원팀 스킨 적용, 응원톡에 팀 로고 표시 |
| **개인화 피드** | 마이팀 경기 일정/결과를 네이버 홈 및 스포츠 탭에 우선 노출 |
| **푸시 알림** | 경기 예고, 경기 결과 등 실시간 알림 |
| **응원톡** | 경기 중 실시간 채팅 (스포츠 뉴스 댓글 폐지 이후 대체 커뮤니티 기능) |
| **라이브 액티비티/다이나믹 아일랜드** | iOS 잠금화면 및 다이나믹 아일랜드에 실시간 경기 스코어 표시 (네이버앱 12.12.0+, iOS 17.2+) |
| **Samsung Now Bar** | OneUI7 기반 갤럭시 기기에서 잠금화면 실시간 스코어 표시 |
| **KBO 구단별 오픈톡** | 10개 구단별 공식 오픈톡방 운영, 실시간 응원 커뮤니티 |
| **클립(숏폼)** | 스포츠 카테고리 숏폼 콘텐츠 생산 및 소비 |

#### 핵심 수치 (2026년 4월 기준)

- **MY팀 누적 이용자 수**: 약 **336만 명** (KBO 10개 구단 합산)
- **마이티켓 발급량**: **61만 건** 돌파 (2026.04.01 기준)
- **KBO 구단별 오픈톡 누적 방문**: 약 **1,273만 건**
- 야구 관련 오픈톡 방문자 수: 개막 2주 전 대비 **246% 증가** (2026.03.28~03.31)
- 스포츠 카테고리 클립 일 평균 생산량: 이전 동기 대비 **30% 증가**

#### 과거 비교 수치

- 2023년 KBO 경기당 최고 동시접속자 평균: **6만 1,000명** (전년 대비 12.6% 증가)
- 2023년 경기당 누적 재생 평균: 약 **61만 회** (전년 대비 27.5% 증가)
- 2023년 한국시리즈 3차전(LG vs kt): 최고 동접 **약 41만 명**, 누적 재생 **약 291만 회**
- 5년간 누적 시청자 수: **약 8억 명** (2024년 1월 보도)

### 1.2 사용자 경험 (UX)

#### 마이팀 등록 경로 (앱 기준)

1. 네이버앱 실행 > 우측 상단 메뉴(삼줄) > 전체 서비스 > **"스포츠"** 선택
2. 스포츠 페이지 내 종목 선택 (야구, 축구, 농구, 배구 등)
3. 응원할 구단 옆 **"+" 버튼** 클릭 > **저장**
4. 해당 팀 페이지에서 **"알림 설정"** 활성화

#### 웹과 앱 인터페이스 비교

| 항목 | 네이버 앱 | 웹 (m.sports.naver.com / sports.naver.com) |
|------|-----------|-------------------------------------------|
| MY팀 등록 | 지원 (메뉴 > 스포츠 > 팀 추가) | 지원 (스포츠 홈에서 팀 추가) |
| 푸시 알림 | 지원 (경기 예고, 결과, 구단 알림) | 미지원 (웹은 푸시 불가) |
| 라이브 액티비티 | iOS/Android(OneUI7) 지원 | 해당 없음 |
| 응원톡 | 앱 내 실시간 채팅 | 웹에서도 접근 가능 |
| 위젯 | 홈 화면 위젯 지원 | 해당 없음 |
| Chrome 확장 | 해당 없음 | Chrome 확장 프로그램 제공 (경기 스코어 확인) |

#### 네이버 홈 노출 방식

- MY팀 등록 시 네이버 메인(홈) 상단 또는 관심사 영역에 해당 팀 경기 일정/스코어가 **우선 노출**
- 스포츠 탭 진입 시 MY팀 경기가 최상단에 배치

### 1.3 알림 체계

알림을 정상 수신하기 위해서는 **다단계 설정**이 필요하다:

1. **네이버 설정** > 알림 수신 설정 > 전체 푸시 알림 **ON**
2. **네이버앱 설정** > 스포츠 알림 > 푸시 알림 **ON**
3. **네이버앱 설정** > 스포츠 알림 > 구단 알림받기 목록 > MY팀 알림 **ON**
4. **네이버앱 설정** > 스포츠 알림 > 관심 구단 **경기예고**, **경기결과** ON
5. (iOS) 설정 > 앱 > NAVER > 실시간 현황 > 실시간 허용 **ON**

#### 알림 종류

| 알림 유형 | 설명 | 비고 |
|-----------|------|------|
| 경기 예고 | 경기 시작 전 사전 알림 | 지원 확인됨 |
| 경기 결과 | 경기 종료 후 스코어 알림 | 지원 확인됨 |
| 구단 알림 | MY팀 관련 뉴스/이벤트 | 지원 확인됨 |
| 실시간 스코어 | 잠금화면/다이나믹 아일랜드 라이브 표시 | iOS 17.2+, OneUI7+ |
| 실시간 득점 알림 | 이닝별/득점 시 실시간 푸시 | [지식 기반, 검색 불가] 세부 세분화 수준 미확인 |

> **참고**: 알림 설정이 다단계로 나뉘어 있어 초기 설정이 복잡하다는 사용자 불만이 SNS에서 관찰됨. Threads, TikTok 등에 "알림 설정 가이드"가 다수 게시되어 있어, 설정 과정의 UX 개선 필요성을 시사.

### 1.4 지원 종목 현황

| 종목 | 리그 | MY팀 설정 지원 | 비고 |
|------|------|----------------|------|
| **야구** | KBO | **지원** | 가장 활발. 마이티켓, 응원톡, 오픈톡 등 부가기능 풍부 |
| **농구** | KBL (남자) | **지원** | 마이팀 설정 가능 |
| **배구** | V리그 | **지원** | 마이팀 설정 가능 |
| **축구** | K리그 | **미지원 또는 제한적** | 나무위키 기록 상 축구 마이팀 미지원. 별도 확인 필요 [출처 불명확] |
| **해외축구** | EPL, 라리가 등 | **미지원** | 경기 정보 제공만 |
| **해외농구** | NBA | **미지원** | NBA Mania 커뮤니티에서 사용자 불만 확인됨 |
| **해외야구** | MLB | [지식 기반, 검색 불가] | 미확인 |

> **핵심 이슈**: KBO 중심으로 기능이 고도화되어 있으며, 축구(K리그)와 해외 리그(NBA, EPL 등)의 마이팀 지원이 부재하거나 제한적이라는 점이 사용자 불만의 주요 원인 중 하나.

### 1.5 현재 한계점

#### 서비스 기능 측면

1. **종목 편중**: KBO에 기능이 집중되어 있으며, K리그/NBA/EPL 등 타 종목 사용자 니즈 충족 미흡
2. **알림 설정 복잡성**: 최소 4~5단계의 설정이 필요하여 이탈 가능성 높음
3. **알림 세분화 부족**: "경기 예고"와 "경기 결과" 중심이며, 실시간 득점/이닝별 알림 등 세분화 수준 미확인 [지식 기반, 검색 불가]
4. **댓글 폐지 후유증**: 2020년 8월 스포츠 뉴스 댓글 전면 폐지 이후 응원톡으로 대체했으나, "댓글을 부활시키라"는 비판 지속 (나무위키 기록)
5. **마이티켓 한정 종목**: 마이티켓은 KBO에만 적용, 타 종목 확장 미비

#### UX 측면

6. **네이버 홈 개인화 간섭**: 숏폼(클립) 및 블로그 콘텐츠가 홈에 과도하게 노출되어, 순수 스포츠 정보만 원하는 사용자에게 불편
7. **앱/웹 기능 격차**: 앱 중심 기능(알림, 위젯, 라이브 액티비티)이 많아 웹 사용자 경험이 상대적으로 열위

## 2. 국내 유사 서비스 비교

### 2.1 카카오스포츠/다음스포츠

#### 서비스 개요

다음스포츠(sports.daum.net)는 카카오 콘텐츠CIC가 운영하는 스포츠 정보 플랫폼이다.

#### 2025년 주요 개편 사항

- 2025 프로야구 개막을 앞두고 **야구게임센터** 대폭 개편
- **그래픽 중계** 고도화: 실시간 경기 상황을 그래픽으로 표시
- **필드뷰** 고도화: 경기장 상황 시각적 표현 강화
- 투타 정보, 실시간 투수-타자 매치업, 팀 전력 정보 등 한눈에 확인 가능

#### 팀 구독/팔로우 기능 비교

| 항목 | 네이버 스포츠 | 다음 스포츠 |
|------|--------------|------------|
| 팀 팔로우/구독 기능 | **MY팀** (구독, 알림, 개인화) | **미확인** — 명시적 팀 팔로우 기능 검색 결과 없음 [검색 한계] |
| 푸시 알림 | 경기 예고, 결과, 구단 알림 | 미확인 [검색 한계] |
| 팀 스킨/개인화 | 마이티켓 스킨 적용 | 미지원으로 추정 |
| 실시간 채팅 | 응원톡 | 미확인 |
| 그래픽 중계 | 지원 | 지원 (2025 개편으로 고도화) |
| 라이브 액티비티 | 지원 | 미확인 |

> **분석**: 다음스포츠는 경기 데이터 시각화(그래픽 중계, 필드뷰)에 강점이 있으나, 네이버 마이팀과 같은 명시적 **팀 구독/개인화 시스템**은 확인되지 않았다. 이는 네이버 마이팀의 차별화 포인트이자 경쟁 우위를 나타낸다.

## 3. 사용자 반응

### 3.1 긍정적 반응

- **마이팀 설정 추천 콘텐츠 확산**: Threads, TikTok, Instagram 등 SNS에서 "네이버 마이팀 꼭 설정하세요"류의 추천 게시물이 다수 생산되고 있음. 이는 서비스 만족도와 자발적 확산을 보여주는 긍정 지표.
- **마이티켓의 팬 몰입감**: 팀 스킨 적용과 응원팀 로고가 팬 정체성 표현에 기여한다는 평가.
- **라이브 액티비티 호평**: "구글(앱)은 필요 없어졌습니다" 등의 반응 — iOS/Android 잠금화면 실시간 스코어 기능에 대한 높은 만족도.
- **오픈톡 활성화**: 개막 직후 야구 관련 오픈톡 방문자 수 246% 급증은 커뮤니티 기능의 성공적 정착을 시사.

### 3.2 부정적 반응 및 불만

| 불만 유형 | 내용 | 출처 |
|-----------|------|------|
| **종목 확대 요구** | NBA 마이팀 설정이 없다는 불만 | NBA Mania 커뮤니티 |
| **축구 마이팀 미지원** | K리그/해외축구 팬의 마이팀 부재 불만 | 나무위키 기록 [출처 불명확] |
| **댓글 부재** | 댓글 폐지 이후 팬 간 소통 제한, 응원톡이 댓글을 완전 대체하지 못함 | 나무위키, 커뮤니티 |
| **알림 설정 복잡** | 4~5단계 설정이 필요하여 초보 사용자가 알림을 받지 못하는 경우 빈번 | SNS (Threads, TikTok에 설정 가이드 다수) |
| **홈 피드 간섭** | 숏폼/블로그 콘텐츠가 스포츠 정보와 혼재되어 노출 | 앱스토어 리뷰 |

### 3.3 커뮤니티 동향

- **네이버 카페**: 10개 구단 팬 카페 중심으로 활발. 야구 테마 카페 페이지뷰 2023년 대비 **2배 이상 증가** (2025년 기준).
- **MLB파크**: 하루 게시물 수 기준 '한국야구타운'이 'MLB타운'의 10~20배 수준으로 국내 야구 논의 중심지.
- **디시인사이드**: 각 구단 갤러리에서 마이팀 관련 논의가 간헐적으로 이루어지나, 주요 논의 주제는 아님. [지식 기반, 검색 불가]

## 5. 추가 조사 가능 항목

아래 항목은 본 조사에서 충분히 확인하지 못하여, 추가 조사가 필요한 영역이다.

1. **네이버 스포츠 마이팀 축구(K리그) 지원 여부의 최신 상태**: 나무위키 기록과 실제 앱 상태 간 괴리 확인 필요
2. **알림 세분화 수준의 정확한 스펙**: 이닝별 알림, 선수 이벤트(홈런, 교체 등) 알림의 존재 여부
3. **다음스포츠의 팀 팔로우/알림 기능 유무**: 직접 앱/웹 확인 필요
4. **앱스토어/플레이스토어 리뷰 정량 분석**: "마이팀" 키워드 리뷰 평점 분포 및 주요 불만 유형 정량화
5. **해외 유사 서비스 벤치마킹**: ESPN 앱, Yahoo Sports, The Athletic 등의 팀 팔로우 기능 비교 (별도 리서치로 분리 가능)
6. **SPOTV NOW 및 쿠팡플레이의 팀 개인화 기능 상세 스펙**: 직접 앱 확인 필요
7. **마이팀 구독자 수의 종목별/구단별 분포**: 공개 데이터 부재 — 네이버 내부 데이터 필요
8. **마이팀과 네이버플러스 멤버십 연계 가능성**: 비즈니스 모델 측면의 심층 분석
