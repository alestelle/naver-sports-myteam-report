### 사례 1: MLB Ballpark 앱 - 경기장 체크인 리워드

| 항목 | 내용 |
|------|------|
| **서비스** | MLB Ballpark App (미국) |
| **시기** | 2022~현재 (지속 운영) |
| **내용** | 경기장 도착 시 위치 기반 자동 체크인 → 방문 이력이 "디지털 야구 여권(Baseball Passport)"에 기록 → 방문 횟수별 배지/한정 콘텐츠 해금. 체크인 시 구단별 한정 쿠폰(식음료 할인, 굿즈 할인) 즉시 지급 |
| **성과** | 누적 다운로드 2,100만+, 경기당 앱 활성 사용률 63% 이상 |

**마이팀 적용안:**
- 네이버 앱 위치 서비스 활용, KBO 10개 구장 반경 500m 진입 시 "직관 인증" 팝업 자동 노출
- 팝업 문구: "오늘 [팀명] 직관 왔어요! 마이팀 등록하고 직관 배지 받기"
- 1탭으로 마이팀 등록 + 직관 배지 자동 부여
- 직관 3회 달성 시 네이버페이 포인트 1,000원 지급
- 개발 공수: 위치 기반 팝업 로직 2주, 배지 시스템 1주

| 항목 | 내용 |
|------|------|
| **서비스** | DraftKings, FanDuel (미국) |
| **시기** | 상시 운영 |
| **내용** | 기존 사용자가 고유 추천 링크를 친구에게 공유 → 친구가 가입+첫 활동 완료 시 추천인과 신규 모두에게 보너스 지급. DraftKings: 추천 1건당 최대 $100, FanDuel: 월 최대 5명, 건당 $75 |
| **성과** | FanDuel 월간 추천 전환율 12~18%, 추천 가입자의 30일 잔존율이 일반 가입자 대비 2.3배 |

**마이팀 적용안:**
- 마이팀 등록 완료 화면에 "친구에게 마이팀 추천하기" 공유 버튼 추가
- 카카오톡/문자/링크 복사로 공유 → 친구가 마이팀 등록 시 양쪽 모두 네이버페이 포인트 500원 지급
- 추천 3명 달성 시 추가 보너스 1,000원 (총 2,500원 수령 가능)
- 공유 메시지 예시: "[닉네임]님이 [팀명] 마이팀에 초대했어요! 지금 등록하면 500포인트 🎁"
- 개발 공수: 추천 링크 생성/추적 시스템 2주, 포인트 지급 연동 1주

- 연속 5경기 스트릭 → "열혈팬" 배지, 10경기 → "철인팬" 배지, 전 경기 → "시즌 올클" 배지
- 스트릭 끊어지면 리셋 → 유지 동기 부여, 경기일마다 푸시 알림 "스트릭 유지하세요!"
- 월간 스트릭 랭킹 → 상위 100명 추첨으로 직관 티켓 2매 제공
- 개발 공수: 스트릭 로직 2주, 배지 시스템 기존 마이티켓과 연동 1주

| **목표** | 신규 마이팀 등록 50만 명, SNS 바이럴 노출 1,000만 회 |

### 참여 방법 Step by Step

```
Step 1: 마이팀 등록 (시작점)
├─ 네이버 앱 → 스포츠 → "마이팀 챌린지" 배너 클릭
├─ 응원 팀 선택 (1탭)
├─ 알림 설정 (경기 예고/스코어/결과 3종 중 1개 이상 ON)
└─ → "챌린지 참여 완료!" 화면 + 챌린지 패스 발급

Step 2: 주간 미션 수행 (매주 월요일 갱신)
├─ Week 1: "첫 응원" - 이번 주 경기 중 응원톡에 메시지 1개 이상 작성
├─ Week 2: "예측의 신" - 이번 주 3경기 이상 승패 예측 참여
├─ Week 3: "전파왕" - 마이팀 챌린지를 친구 1명 이상에게 공유
├─ Week 4: "직관 인증" - 경기장 방문 인증 또는 TV 시청 인증(중계 화면 캡처)
├─ Week 5~8: 위 미션 조합 + 보너스 미션(구단별 퀴즈, 역사 퀴즈 등)
└─ → 미션 완료 시 챌린지 패스에 스탬프 적립

Step 3: 보상 수령
├─ 미션별 즉시 보상 (포인트, 배지)
├─ 누적 스탬프별 티어 보상
└─ 시즌 종료 시 최종 랭킹 보상

Step 4: SNS 공유 (바이럴 루프)
├─ 미션 완료/배지 획득 시 "공유하기" 버튼 활성화
├─ 공유 이미지 자동 생성: 팀 로고 + 배지 + 닉네임 + "나는 [팀명] 챌린지 [티어]!"
├─ 카카오톡/인스타 스토리/쓰레드/X 공유 가능
└─ 공유 링크 클릭 시 → 챌린지 참여 페이지로 직행 (신규 유입)
```

## 4. 비야구 종목 프로모션 전략

### 4-1. K리그 (축구) 팬 대상

#### K리그 팬 특성
- KBO 대비 연령층이 다양 (10대~40대 고르게 분포)
- 지역 연고 의식이 강함 (울산, 전북, 수원 등)
- 주말 경기 집중, 평일 관심 상대적으로 낮음
- 글로벌 축구(EPL/라리가)와 병행 시청하는 팬 다수

#### 구체 프로모션

**프로모션 K-1: "K리그 매치데이 알림 서비스"**
- 실행: 네이버 스포츠 K리그 중계 페이지에 "이 팀 경기 놓치지 않기" 플로팅 버튼 → 마이팀 등록
- 문구: "주말 경기 까먹지 마세요! [팀명] 경기 알림 받기"
- 타이밍: 금요일 저녁 푸시 "내일 [팀명] 경기! 마이팀 등록하고 알림 받으세요"
- 차별화: K리그는 주 1~2경기라 알림 피로도 낮음 → "부담 없는 알림"을 강조

**프로모션 K-2: "K리그 더비 대결 이벤트"**
- 실행: 수원 삼성 vs 수원 FC, 서울 vs 인천 등 더비 매치 시 "마이팀 응원수 대결" 배너
- 보상: 승리 팀 마이팀 팬 전원에게 네이버페이 100P (소액이지만 "우리가 이겼다!" 만족감)
- 예상 효과: 더비 매치당 3,000~5,000명 신규 등록

**프로모션 K-3: "EPL 팬 → K리그 마이팀 크로스 프로모션"**
- 실행: 네이버 스포츠 EPL 페이지에서 "좋아하는 해외 리그가 있다면, K리그도 챙겨보세요" 배너
- K리그 소속 전 EPL/해외 출신 선수 하이라이트와 연계
- 문구: "해외 축구 팬이라면 K리그도! 마이팀 등록하고 국내 축구 소식도 받아보세요"

### 4-2. KBL (농구) 팬 대상

#### KBL 팬 특성
- 20~30대 남성 비율 높음
- 인터넷/커뮤니티 활동 활발 (루리웹, 디시 농구갤 등)
- 겨울 스포츠 (10월~4월) → KBO 비시즌과 겹침
- 2024-25 시즌부터 tvN SPORTS/TVING 중계로 접근성 확대

#### 구체 프로모션

**프로모션 B-1: "KBO 비시즌 = KBL 시즌" 크로스 온보딩**
- 실행: 10월 KBO 시즌 종료 후 마이팀 등록자에게 푸시 알림
- 문구: "야구 시즌이 끝났지만 스포츠는 계속! KBL [지역팀] 마이팀 등록하고 겨울에도 응원하세요"
- KBO 마이팀 팀 지역과 동일 연고 KBL 팀 자동 추천 (예: 서울 LG 팬 → 서울 SK 나이츠 추천)
- 예상 효과: KBO 마이팀 등록자의 15%가 KBL 팀 추가 등록 = 약 50만 명

**프로모션 B-2: "올스타전 팬투표 연계"**
- 실행: KBL 올스타전 팬투표를 네이버 마이팀에서 진행
- 투표 참여 조건: KBL 팀 마이팀 등록 필수
- 보상: 투표 참여자 중 추첨 올스타전 티켓 + 선수 사인볼

**프로모션 B-3: "농구 예측왕" 게임**
- 실행: KBL 경기 시작 전 "오늘의 승리팀 + 최다득점자" 예측 게임
- 마이팀 등록 필수, 적중 시 포인트 지급
- 농구 특성 활용: 쿼터별 마이크로 예측 (3점슛 성공 수, 전반 스코어 등)

### 4-3. LCK / e스포츠 팬 대상

#### e스포츠 팬 특성
- 10대 후반~20대 초반 핵심 타겟
- 디지털 네이티브, SNS 활동 매우 활발
- 팀보다 선수 개인 팬덤이 강함 (Faker, Chovy 등)
- 치지직/SOOP 중심 시청, 네이버 스포츠 접점 상대적으로 약함
- 밈(meme) 문화에 친숙, 게이미피케이션 반응도 높음

#### 구체 프로모션

**프로모션 E-1: "LCK 픽 챌린지"**
- 실행: LCK 매 경기 시작 전 "오늘의 승리팀 + MVP" 예측 게임
- 네이버 스포츠 e스포츠 페이지 최상단에 예측 카드 노출
- 참여 조건: LCK 팀 마이팀 등록 필수
- 보상: 주간 적중률 1위 RP(Riot Points) 50,000원 상당 + "예측의 신" 칭호
- 차별화: LoL 인게임 보상(RP)과 연계하여 e스포츠 팬 특화 리워드 제공

**프로모션 E-2: "치지직/SOOP → 네이버 마이팀" 크로스 유도**
- 실행: 네이버 계열 치지직에서 LCK 중계 시 화면 하단에 "네이버 마이팀에서 [팀명] 팬 활동하기" 배너
- QR/링크로 네이버 스포츠 마이팀 등록 페이지로 이동
- 마이팀 등록 시 치지직 구독 배지와 연동되는 "LCK 팬 인증" 이모티콘 제공
- 예상 효과: 치지직 LCK 동시접속 10만명 × 전환율 3% = 3,000명/경기

**프로모션 E-3: "선수 밈 투표" 바이럴 이벤트**
- 실행: 매주 "이번 주 최고의 LCK 순간" 밈 투표 (하이라이트 플레이 3개 후보)
- 네이버 스포츠 마이팀 페이지에서 투표, 결과를 GIF/숏폼으로 자동 생성 → SNS 공유
- e스포츠 팬의 밈 문화 활용: "그 장면 보고 마이팀 등록 안 할 수 있어?" 톤의 밈형 콘텐츠
- 예상 효과: 투표 참여 주 5만명, SNS 공유 1만건

**프로모션 E-4: "마이팀 X LCK 시즌 패스"**
- 실행: LCK 시즌 시작 시 "마이팀 시즌 패스" 발급 (무료)
- 패스 보유 시 독점 콘텐츠 접근: 선수 인터뷰 비하인드, 팀 연습실 사진, 선수 Q&A 참여권
- 시즌 종료 시 패스 보유 기간에 따라 등급 부여 → "시즌 컴플리트" 배지
- 10대 타겟 특성상 "디지털 수집" 욕구를 자극하는 배지/카드 시스템이 효과적
