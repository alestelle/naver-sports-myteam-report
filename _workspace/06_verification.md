## 1. K리그 마이팀 지원 여부

### 결과: 지원 (확인됨)

네이버 스포츠에서 K리그(축구) 마이팀 설정이 **가능하다**.

- **K리그1, K리그2 구단별 마이팀 추가 가능**: 네이버 스포츠의 마이팀 지원 종목에 "프로축구(K리그1, K리그2)"가 명시적으로 포함되어 있음
- **지원 종목 전체 목록**: 야구(KBO, 해외야구), 축구(K리그1, K리그2, 해외축구), 농구(남자/여자프로농구, NBA), 배구(남자/여자), 골프, e스포츠
- **마이팀 최대 등록 수**: 최대 9개 팀까지 지정 가능
- **서비스 시작 시기**: 정확한 시작 시점은 공식 발표 자료에서 확인 불가. 다만 네이버 스포츠 서비스 자체가 오래 전부터 운영되었으며, 마이팀 기능은 현재 안정적으로 제공 중

## 2. MLB 마이팀 지원 여부

### 결과: 지원 (확인됨)

네이버 스포츠에서 MLB 팀 마이팀 설정이 **가능하다**.

- **해외야구 카테고리**에서 MLB(메이저 리그 베이스볼), 일본프로야구(NPB), 해외야구 기타를 지원
- MLB 30개 팀을 마이팀으로 추가할 수 있으며, 다른 종목 팀과 합쳐서 최대 9개까지 등록 가능
- 한국인 MLB 선수 관련 별도 기능(선수별 알림 등)에 대해서는 명확한 근거를 찾지 못함. 다만 네이버 스포츠에서 한국인 MLB 선수 관련 뉴스와 경기 정보는 별도 섹션에서 제공하는 것으로 추정

### 근거
- [나무위키 - 네이버 스포츠](https://namu.wiki/w/%EB%84%A4%EC%9D%B4%EB%B2%84%20%EC%8A%A4%ED%8F%AC%EC%B8%A0): "해외야구 종목으로 메이저 리그 베이스볼, 일본프로야구, 해외야구 기타를 지원"
- 웹 검색 종합: MLB 팀을 마이팀으로 설정하여 경기 시작 1시간 전 알림, 경기 결과, 하이라이트 알림을 받을 수 있음

## 3. 네이버 스포츠 알림 세분화 정확한 스펙

### 결과: 부분 확인

확인된 알림 종류는 다음과 같다:

#### 푸시 알림 (기본)
1. **경기 예고 알림** - 경기 시작 약 1시간 전 알림
2. **경기 결과 알림** - 경기 종료 후 결과 알림
3. **하이라이트 알림** - 경기 하이라이트 영상 등록 시 알림

#### 실시간 현황 (Live Activity)
4. **잠금화면/다이나믹 아일랜드 실시간 스코어** - 경기 중 실시간 점수 표시
   - iOS: 네이버앱 12.12.0 이상 + iOS 17.2 이상 필요 (다이나믹 아일랜드는 iPhone 14 Pro 이상)
   - Android: ONE UI 7 + Now Bar 지원 기종 (Samsung Galaxy S25 등)

#### 알림 설정 경로
- 네이버앱 설정 > 알림 수신 설정 > 전체 푸시 알림 ON
- 네이버앱 설정 > 스포츠 알림 > 푸시 알림 ON
- 네이버앱 설정 > 스포츠 알림 > 구단 알림받기 목록 > MY팀 알림 ON
- 네이버앱 설정 > 스포츠 알림 > 관심 구단 경기예고, 경기결과 ON

#### 확인 불가 항목
- **실시간 득점 알림(푸시)**: 별도의 득점 시 푸시 알림이 있는지 웹 검색으로 명확히 확인 불가. Live Activity를 통해 실시간 스코어는 확인 가능하나, 득점 시마다 별도 푸시가 오는지는 불분명
- **이닝별 알림**: 이닝 종료/시작 시 알림이 오는지 확인 불가
- **선수별 알림(홈런, 타석 등)**: 선수 단위의 세분화된 알림은 확인되지 않음. 네이버 스포츠에서 제공하는 알림은 팀(구단) 단위가 기본

## 4. 다음스포츠 팀 팔로우 기능

### 결과: 미지원 (해당 기능 없음으로 판단)

다음(Daum) 스포츠에는 네이버 스포츠의 "마이팀"에 해당하는 **팀 팔로우/구독/즐겨찾기 기능이 없는 것으로 판단**된다.

- 2025년 3월 카카오가 '다음 스포츠 야구게임센터'를 대대적으로 개편했으나, 개편 내용에 팀 구독/팔로우/알림 기능은 포함되지 않음
- 개편 내용은 그래픽 중계 고도화, 필드뷰, AI 경기 요약, 비더레전드(안타예측 이벤트), MVP 투표, 구단 응원 투표, 구단별 다음카페 테이블 연동 등에 집중
- 팀 단위 푸시 알림이나 개인화된 팀 피드 기능에 대한 언급은 카카오 공식 보도자료에도, 관련 뉴스 기사에도 없음
- 다음 스포츠에는 경기 일정, 순위, 팀 정보 페이지는 있으나 개인화된 "내 팀" 설정 기능은 확인되지 않음
