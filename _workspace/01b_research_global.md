## 1. ESPN -- My ESPN 개인화

### 1.1 팀 팔로우/즐겨찾기 구현 방식

- **MyESPN 탭**: 사용자가 원하는 스포츠 종목을 선택하고 표시 순서를 지정하며, 즐겨찾기 팀(Favorite Teams)을 등록하여 최신 팀 뉴스를 수신하고 스코어보드에서 해당 팀을 우선 표시
- **팔로우 대상**: 팀, 리그, 선수, 스포츠 종목 단위로 팔로우 가능
- **크로스 플랫폼**: ESPN 앱, ESPN.com, Disney+ 내 ESPN 탭에서 동일한 팔로우 설정 공유

### 1.2 개인화 피드 작동 방식

- **SC For You (SportsCenter For You)**: 2025년 8월 DTC 출시와 함께 도입. AI 기반으로 사용자가 팔로우한 팀/리그/종목에 맞춘 하이라이트, 뉴스, 필수 영상을 매일 제공. AI 생성 앵커 보이스오버(Hannah Storm, Gary Striewski 등)를 사용한 개인화 스포츠센터
- **ESPN Verts**: 틱톡/릴스 스타일의 세로형 숏폼 영상 피드. 고급 개인화 알고리즘 기반으로 사용자 관심사에 맞는 영상을 스와이프 방식으로 제공
- **개인화 기술**: Disney Entertainment와 공동 개발한 추천 알고리즘. 명시적(팔로우한 팀) + 암묵적(시청 행태) 개인화를 결합
- **ESPN StreamCenter**: 실시간 경기와 동기화된 세컨드 스크린 경험. 실시간 게임 통계, 판타지 스탯, ESPN BET 정보 통합 표시

### 1.3 알림 세분화 수준

| 알림 유형 | 설명 |
|-----------|------|
| Score Updates | 경기 중 득점 업데이트 |
| Game Start | 경기 시작 알림 |
| Breaking News | 속보 (트레이드, FA, 부상 등) |
| Reporter Alerts | ESPN 기자의 독점 속보 알림 (2025년 2월 도입) |
| Gambling | 베팅 관련 알림 |
| Instant Awesome | 주요 하이라이트/명장면 알림 |
| Fantasy Alerts | 트레이드, Add/Drop, 홈런 등 판타지 관련 세부 알림 |

- 설정 방법: 앱 우측 상단 톱니바퀴 > Alert Preferences에서 카테고리별 토글 ON/OFF

### 1.4 구독자 유지 기능

- **독점 콘텐츠**: SC For You는 ESPN Unlimited 및 케이블 가입자 전용
- **Catch Up To Live**: 실시간 경기 도중 놓친 부분을 하이라이트로 빠르게 따라잡기
- **멀티뷰**: 커넥티드 TV에서 최대 4경기 동시 시청
- **판타지/베팅 통합**: 경기 시청 중 판타지 결과, 베팅 정보를 실시간으로 표시
- **Fanatics 커머스**: QR코드를 통한 실시간 경기 관련 상품 구매
- **가격 체계**: Unlimited $29.99/월, Disney+/Hulu 번들 $29.99/월(12개월), Fox One 번들 $39.99/월

## 2. Yahoo Sports

### 2.1 팀 구독 기능의 특징

- **My Teams & Leagues**: 앱 내 More > Edit에서 좋아하는 팀과 리그를 등록
- **구독 시 제공 콘텐츠**: 브레이킹 뉴스, 일정 리마인더, 하이라이트, 경기 통계, 다가오는 경기 목록을 큐레이션하여 제공
- **커버리지 범위**: NFL, NBA, MLB, NHL, NCAA, WNBA, 축구 등 주요 프로/대학 리그 전체

### 2.2 판타지 스포츠와의 연계

- **통합 앱 경험**: 2024년 앱 대규모 리디자인을 통해 기존 2~3개 앱에 분산되어 있던 기능을 단일 앱으로 통합. Home, News, Scores 3개 탭으로 원클릭 전환
- **실시간 판타지 연동**: 실시간 라이브 스코어가 판타지 매치업 점수에 즉각 반영
- **Draft Kit**: 사용자의 리그 세팅에 맞춘 개인화 치트시트, 랭킹, 티어, 예측 제공
- **커스터마이징**: 100개 이상의 새 팀 로고로 판타지 팀 개인화 가능

### 2.3 개인화 뉴스피드 구성 방식

- **큐레이션 피드**: 팔로우한 팀 기반으로 뉴스, 하이라이트, 경기 일정, 통계를 자동 구성
- **알림 세분화**: 경기 시작, 스코어 변동, 부상 뉴스, 빅 모먼트 등 개별 알림 설정 가능
- **알림 레벨 선택**: 속보 플래시부터 일일 다이제스트까지 원하는 수준 선택
- **Lock Screen 통합**: Live Activities를 통해 잠금 화면에서 실시간 경기 정보 및 Key Plays 확인

## 3. The Athletic

### 3.1 팀/리그 구독 모델 (유료)

- **소유 구조**: 2022년 1월 뉴욕타임스(NYT)가 5.5억 달러에 인수
- **구독 모델**: 순수 구독 기반 수익 모델 (광고 의존 안 함). NYT 올-액세스 번들 가입 시 The Athletic 포함 접근 가능 (Games, Cooking, Wirecutter와 함께)
- **콘텐츠 규모**: 450명 이상의 기자, 월 4,000건 이상 기사, 1,000개+ 팀, 40개+ 리그, NCAA 커버
- **팟캐스트**: 80개 이상의 독점 팟캐스트 (일간/주간/리그별). 2025년 8월 Acast와 독점 세일즈 파트너십

### 3.2 구독 전환을 유도하는 팀 팔로우 전략

- **팀/리그/컵 팔로우**: 팔로우한 팀의 뉴스, 스코어, 통계를 한곳에서 확인 가능
- **로컬 비트 리포터**: 100명 이상의 팀 전담 기자를 배치하여 47개 북미 도시 + 영국에서 로컬 커버리지 제공 -- 이것이 핵심 차별화 포인트
- **개인화 피드의 한계**: 일부 사용자들은 팔로우한 팀 콘텐츠만 보이는 완전한 개인화 피드를 원하지만, 현재는 관심 없는 콘텐츠도 혼재되는 것으로 보고됨

### 3.3 팬 인게이지먼트 수치

- **구독자 수**: 2024년 말 기준 583만 명이 The Athletic에 접근 (직접 구독 + 번들 구독 포함)
- **알림**: 축구, 테니스, F1, 크리켓 등 실시간 업데이트 알림 지원

## 5. OneFootball

### 5.1 클럽 팔로우 기능의 특징

- **Top News 탭**: 팔로우 여부와 무관하게 지난 24시간 주요 글로벌 축구 뉴스 표시

|-----------|------|
| Goal Alerts | 골 알림 |
| Kickoff | 경기 시작 알림 |
| Match Result | 경기 결과 알림 |
| Lineup | 라인업 발표 알림 |

### 6.3 Yahoo! JAPAN 스포츠나비 (Sports Navi)

- **포지셔닝**: 일본 최대 스포츠 포털. 프로야구(NPB) 중심이며 네이버 스포츠와 가장 유사한 비교 대상
- **팔로우 기능 전환 (2024년 9월)**: 기존 단일 "즐겨찾기 팀" 방식에서 복수 팀 "팔로우" 방식으로 변경
- **팔로우 대상 확대**:
  - NPB 12개 구단 + 팜리그 2개 팀
  - MLB 30개 팀 전체
  - 고교야구, 대학야구 팀
- **선수 팔로우**: NPB 전 선수, MLB 일본인 선수, 일본 국가대표 선수 개별 팔로우 가능
- **선수 알림 세분화**: 선발 라인업, 타석, 홈런, 1군 등록/말소 등 세부 이벤트별 알림
- **팀 알림**: 팀별로 알림 카테고리 개별 ON/OFF 가능 (1군/2군 분리)
- **크로스 플랫폼**: Yahoo! JAPAN 테마와 Sports Navi 간 팔로우 정보 자동 동기화

### 7.3 글로벌 스탠다드 vs 네이버 마이팀 격차

#### 네이버 스포츠 마이팀 현황 요약
- 최대 9개 팀 MY팀 등록 가능
- 경기 시작 1시간 전 알림, 결과/하이라이트 알림
- 마이티켓으로 응원팀 스킨 적용 + 응원톡 로고 생성
- 누적 이용자 약 336만 명 (10개 KBO 구단 기준)
- 응원톡/오픈톡 등 커뮤니티 기능 연계

#### 주요 격차 분석

| 영역 | 글로벌 스탠다드 | 네이버 마이팀 현황 | 격차 수준 |

#### 네이버 마이팀의 상대적 강점

1. **실시간 응원톡/오픈톡**: 경기 중 실시간 팬 채팅은 글로벌 서비스 대비 차별화 포인트. ESPN StreamCenter가 유사하나, 네이버의 톡 기반 커뮤니티가 더 활발
2. **마이티켓 시스템**: 팀 스킨/응원 로고 등 시각적 팬 아이덴티티 기능은 독특한 요소
3. **무료 기반 대규모 사용자**: 336만 누적 이용자로 한국 시장에서 압도적 점유율
4. **네이버 생태계 연계**: 검색, 뉴스, 카페, 클립 등 네이버 전체 생태계와의 연동 잠재력
