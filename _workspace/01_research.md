# 통합 리서치: 네이버 스포츠 마이팀 현황 분석 및 구독자 증가 전략

> **조사 기준일**: 2026-04-08  
> **목적**: 내부 전략팀 보고서 — 네이버 스포츠 마이팀 구독자 수 증가 및 서비스 개선 방안 도출  
> **통합 범위**: 국내 서비스 현황, 글로벌 벤치마킹, 구독 전략 리서치 3건 병합

---

## 핵심 발견사항 요약

### 1. 네이버 스포츠 마이팀 현황
- **MY팀 누적 이용자 약 336만 명** (KBO 10개 구단 합산, 2026.04 기준), 마이티켓 발급량 61만 건 돌파
- KBO 구단별 오픈톡 누적 방문 약 1,273만 건, 개막 후 방문자 수 246% 급증
- **종목 편중 문제**: KBO 중심 기능 고도화, K리그/NBA/EPL 마이팀 미지원 또는 제한적
- **알림 설정 복잡성**: 최소 4~5단계 설정 필요 — 사용자 이탈 요인
- 무료 기반 모델로 진입 장벽 낮으나 수익화 구조 약함

### 2. 글로벌 벤치마킹 핵심 격차
- **AI 개인화**: ESPN SC For You, Google Gemini 스포츠 브리프 등 AI 기반 맞춤 콘텐츠가 글로벌 표준 — 네이버는 팀별 뉴스 목록 수준 (큰 격차)
- **알림 세분화**: ESPN/OneFootball 7종+, Yahoo JP 선수 단위 타석/홈런 알림 — 네이버는 경기 시작/결과 중심 (큰 격차)
- **선수 팔로우**: Yahoo JP 전 선수 팔로우, OneFootball 선수별 뉴스 — 네이버는 팀 단위만 (큰 격차)
- **네이버 강점**: 응원톡/오픈톡 실시간 커뮤니티, 마이티켓 팬 정체성, 무료 대규모 사용자, 네이버 생태계 연계

### 3. 구독자 증가 전략 핵심
- **온보딩 최적화**: 개인화 온보딩 경로는 완료율 35% 향상, 인터랙티브 온보딩은 활성화율 50% 향상
- **게이미피케이션**: 로열티 프로그램 참여 팬의 반복 참여 가능성 약 80% 증가
- **바이럴 루프**: 공유 가능한 콘텐츠(경기 결과 카드, 시즌 래밍) + 이중 보상 친구 초대가 효과적
- **시즌 연계**: 시즌 개막이 최대 기회 — 2026 개막 후 오픈톡 방문 246% 증가 입증
- **구독 피로 방지**: 다이제스트 알림 시 참여율 35% 상승, 옵트아웃율 28% 감소

## Part 1: 네이버 스포츠 마이팀 현황 및 국내 서비스 비교

### 1.1 핵심 기능

네이버 스포츠 마이팀(MY팀)은 네이버앱 및 웹에서 사용자가 응원하는 스포츠 구단을 등록하고, 해당 팀의 경기 일정/결과/뉴스를 우선적으로 제공받는 개인화 기능이다.

#### 주요 기능 목록

| 기능 | 설명 |
|------|------|
| **MY팀 등록** | 종목별 응원 구단을 "+" 버튼으로 추가, 복수 팀 등록 가능 |
| **마이티켓** | 야구 팬 인증 기능. 발급 시 경기 페이지에 응원팀 스킨 적용, 응원톡에 팀 로고 표시 |
| **개인화 피드** | 마이팀 경기 일정/결과를 네이버 홈 및 스포츠 탭에 우선 노출 |
| **푸시 알림** | 경기 예고, 경기 결과 등 실시간 알림 |
| **응원톡** | 경기 중 실시간 채팅 (스포츠 뉴스 댓글 폐지 이후 대체 커뮤니티 기능) |
| **라이브 액티비티/다이나믹 아일랜드** | iOS 잠금화면 및 다이나믹 아일랜드에 실시간 경기 스코어 표시 (네이버앱 12.12.0+, iOS 17.2+) |
| **Samsung Now Bar** | OneUI7 기반 갤럭시 기기에서 잠금화면 실시간 스코어 표시 |
| **KBO 구단별 오픈톡** | 10개 구단별 공식 오픈톡방 운영, 실시간 응원 커뮤니티 |
| **클립(숏폼)** | 스포츠 카테고리 숏폼 콘텐츠 생산 및 소비 |

#### 핵심 수치 (2026년 4월 기준)

- **MY팀 누적 이용자 수**: 약 **336만 명** (KBO 10개 구단 합산)
- **마이티켓 발급량**: **61만 건** 돌파 (2026.04.01 기준)
- **KBO 구단별 오픈톡 누적 방문**: 약 **1,273만 건**
- 야구 관련 오픈톡 방문자 수: 개막 2주 전 대비 **246% 증가** (2026.03.28~03.31)
- 스포츠 카테고리 클립 일 평균 생산량: 이전 동기 대비 **30% 증가**

#### 과거 비교 수치

- 2023년 KBO 경기당 최고 동시접속자 평균: **6만 1,000명** (전년 대비 12.6% 증가)
- 2023년 경기당 누적 재생 평균: 약 **61만 회** (전년 대비 27.5% 증가)
- 2023년 한국시리즈 3차전(LG vs kt): 최고 동접 **약 41만 명**, 누적 재생 **약 291만 회**
- 5년간 누적 시청자 수: **약 8억 명** (2024년 1월 보도)

### 1.2 사용자 경험 (UX)

#### 마이팀 등록 경로 (앱 기준)

1. 네이버앱 실행 > 우측 상단 메뉴(삼줄) > 전체 서비스 > **"스포츠"** 선택
2. 스포츠 페이지 내 종목 선택 (야구, 축구, 농구, 배구 등)
3. 응원할 구단 옆 **"+" 버튼** 클릭 > **저장**
4. 해당 팀 페이지에서 **"알림 설정"** 활성화

#### 웹과 앱 인터페이스 비교

| 항목 | 네이버 앱 | 웹 |
|------|-----------|-----|
| MY팀 등록 | 지원 (메뉴 > 스포츠 > 팀 추가) | 지원 (스포츠 홈에서 팀 추가) |
| 푸시 알림 | 지원 (경기 예고, 결과, 구단 알림) | 미지원 (웹은 푸시 불가) |
| 라이브 액티비티 | iOS/Android(OneUI7) 지원 | 해당 없음 |
| 응원톡 | 앱 내 실시간 채팅 | 웹에서도 접근 가능 |
| 위젯 | 홈 화면 위젯 지원 | 해당 없음 |
| Chrome 확장 | 해당 없음 | Chrome 확장 프로그램 제공 |

### 1.3 알림 체계

알림을 정상 수신하기 위해서는 **다단계 설정**이 필요하다:

1. 네이버 설정 > 알림 수신 설정 > 전체 푸시 알림 ON
2. 네이버앱 설정 > 스포츠 알림 > 푸시 알림 ON
3. 네이버앱 설정 > 스포츠 알림 > 구단 알림받기 목록 > MY팀 알림 ON
4. 네이버앱 설정 > 스포츠 알림 > 관심 구단 경기예고, 경기결과 ON
5. (iOS) 설정 > 앱 > NAVER > 실시간 현황 > 실시간 허용 ON

| 알림 유형 | 설명 | 비고 |
|-----------|------|------|
| 경기 예고 | 경기 시작 전 사전 알림 | 지원 확인됨 |
| 경기 결과 | 경기 종료 후 스코어 알림 | 지원 확인됨 |
| 구단 알림 | MY팀 관련 뉴스/이벤트 | 지원 확인됨 |
| 실시간 스코어 | 잠금화면/다이나믹 아일랜드 라이브 표시 | iOS 17.2+, OneUI7+ |
| 실시간 득점 알림 | 이닝별/득점 시 실시간 푸시 | 세부 세분화 수준 미확인 [검증 필요] |

> **참고**: 알림 설정이 다단계로 나뉘어 있어 초기 설정이 복잡하다는 사용자 불만이 SNS에서 관찰됨.

### 1.5 현재 한계점

#### 서비스 기능 측면
1. **종목 편중**: KBO에 기능 집중, K리그/NBA/EPL 등 타 종목 사용자 니즈 충족 미흡
2. **알림 설정 복잡성**: 최소 4~5단계 설정 필요 — 이탈 가능성 높음
3. **알림 세분화 부족**: 경기 예고/결과 중심, 실시간 득점/이닝별 알림 세분화 미확인
4. **댓글 폐지 후유증**: 2020년 8월 스포츠 뉴스 댓글 전면 폐지 후 응원톡으로 대체했으나 비판 지속
5. **마이티켓 한정 종목**: KBO에만 적용, 타 종목 확장 미비

#### UX 측면
6. **네이버 홈 개인화 간섭**: 숏폼(클립)/블로그 콘텐츠 과도 노출로 순수 스포츠 정보 사용자 불편
7. **앱/웹 기능 격차**: 앱 중심 기능이 많아 웹 사용자 경험 열위

### 1.6 사용자 반응

#### 긍정적 반응

- 2025 프로야구 개막 대비 **야구게임센터** 대폭 개편 (그래픽 중계, 필드뷰 고도화)
- 네이버 마이팀과 같은 명시적 **팀 구독/개인화 시스템** 미확인 — 네이버의 차별화 포인트

### 2.2 SPOTV NOW
- 유료 스포츠 스트리밍. MY팀/리그 설정 후 관련 경기 모아보기, 경기 시작 알림 지원
- 해외 리그(EPL, 분데스리가, UFC 등) 중심. 중계 자체가 핵심 가치

### 2.3 쿠팡플레이
- 쿠팡 와우 멤버십 기반 OTT + 스포츠 스트리밍 (별도 스포츠 패스)

## Part 3: 해외 스포츠 구독 서비스 벤치마킹

### 3.1 ESPN — My ESPN 개인화

- **MyESPN 탭**: 팀, 리그, 선수, 스포츠 종목 단위로 팔로우. 크로스 플랫폼(앱, 웹, Disney+ ESPN 탭) 설정 공유

### 4.1 구독 서비스의 핵심 가치 프레임워크

| **경험적 가치** | 맞춤형 콘텐츠, 개인화된 피드 경험 | MY팀 중심 피드, 맞춤 하이라이트 |

### 4.2 스포츠 팬 구독의 특수성

- 일반 콘텐츠 구독과 달리 **"정체성"에 기반** — 콘텐츠 양/질 개선만으로는 한계
- 실시간성 극도로 중요, 감정 강도 매우 높음, 시즌성 극단적
- 팀 성적 부진 시에도 유지 경향 (진성 팬) — 팬심 기반 충성도

#### 팬 5대 참여 동기 [출처: KCI]
1. 정보 추구 (실시간 경기 정보, 선수 뉴스)
2. 오락 추구 (경기 관람 즐거움 확장)
3. 유니크한 경험 추구 (비하인드, 독점 콘텐츠)
4. 우정 추구 (같은 팀 팬과의 교류)
5. 가치 공유 (팬으로서의 자기표현)

#### 팬이 가장 가치 있게 여기는 정보 [출처: Stats Perform 2026]
- 실시간 경기 업데이트: **35%** (1순위)
- 개인화된 콘텐츠: **30%** (2순위)
- AI 강화 스포츠 경험에 85%가 가치, 63%가 AI 생성 콘텐츠 신뢰

### 4.3 온보딩 & 첫 구독 유도 전략

1. **기본값(Default) 설정**: Opt-out 방식 전환율 48.8% vs Opt-in 18.2% (2.7배 차이) [출처: Phiture]
2. **개인화 온보딩**: 완료율 35% 향상 [출처: UserGuiding]
3. **인터랙티브 온보딩**: 정적 튜토리얼 대비 활성화율 50% 향상 [출처: UserGuiding]
4. **점진적 정보 공개**: 처음부터 모든 설정을 보여주지 않고 사용 맥락에 맞춰 제안
5. **넛지 기법**: 경기 페이지 방문 시 "이 팀의 다음 경기 알림 받기" 원탭 구독 유도 [출처: Nudge]
